## ğŸš€ **Cara Menjalankan di Lokal**

### Prasyarat

- Node.js 18+ dan npm/yarn/pnpm
- Akun Supabase (gratis)
- Gemini API (ngrok)

### Langkah-langkah

1. **Clone repository**

```bash
git clone https://github.com/username/bizness.git
cd bizness
```

2. **Install dependencies**

```bash
npm install
# atau
yarn install
# atau
pnpm install
```

3. **Setup environment variables**

```bash
cp .env.example .env.local
```

Isi variabel di `.env.local`:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key

```

4. **Setup database Supabase**

- Buat project baru di Supabase
- Import skema database dari `/supabase/schema.sql`
- Konfigurasi authentication providers

5. **Jalankan development server**

```bash
npm run dev
# atau
yarn dev
# atau
pnpm dev
```

6. **Buka di browser**

```
http://localhost:8000
```
